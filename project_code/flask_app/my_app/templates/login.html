Welcome to Connections

The purpose of this application is the intelligent and graphical joining of tables. With the Connections Project you can easily visualize all tables and, theoretically, find connections between data sets that aren't entirely obvious. You, the user, can explore yourself with Exploration mode or you can let our computers do the work with our Computer-Guided Learning.

This project was created by Alex Ball, Brian Sandman, Charley Lewittes, Jeremy Choi, Joseph Dyer, Kara Isakson, Nati Abebe, Preston Haynes, Shehan Jayasinghe, and Tai Zheng as part of the University of Maryland's Software Engineering course.

This application is indented to be used by University of Maryland students, faculty and staff. If you not do not have valid credentials, please contact the developers to request access.
